# Privacy Policy for 栈查查 (zhanStack)

**Last Updated: January 12, 2026**

This Privacy Policy describes how "栈查查" (also known as "zhanStack", referred to as "the Extension") collects, uses, and handles your information. We are committed to protecting your privacy and ensuring a transparent user experience.

---

## 1. Data Collection and Usage

**The Extension does not collect, store, or transmit any personal information.**

- **Analysis Data**: To provide its core functionality (analyzing GitHub repository tech stacks), the Extension reads the URL of the active GitHub tab and fetches file metadata and dependency files (e.g., `package.json`, `go.mod`, `requirements.txt`) from GitHub. 
- **Local Processing**: All data extraction and analysis are performed **locally** within your browser. No data is sent to any external servers or third parties.
- **No Personal Data**: We do not collect names, email addresses, browsing history (outside of the active GitHub tab), or any other personally identifiable information (PII).

## 2. Permissions

To function correctly, the Extension requires the following permissions:

- **storage**: To store your local settings and preferences (e.g., display language, UI settings) within your browser. This data is synced across your Chrome devices if you have sync enabled, but it is never shared with us.
- **Host Permissions (`*://github.com/*`, `*://raw.githubusercontent.com/*`)**: To fetch necessary repository metadata and file contents from GitHub's official domains.

## 3. Data Security

Since the Extension does not collect or transmit data to external servers, your data remains secure within your local browser environment. We do not have access to any of the repositories you analyze or your browsing activity.

## 4. Limited Use Disclosure

The Extension's use and transfer to any other app of information received from Google APIs will adhere to the [Chrome Web Store User Data Policy](https://developer.chrome.com/docs/webstore/program-policies/user-data-faq), including the **Limited Use** requirements.

## 5. Changes to This Policy

We may update this Privacy Policy from time to time to reflect changes in our practices or for legal reasons. The "Last Updated" date at the top will indicate when the latest changes were made.

## 6. Contact Us

If you have any questions or suggestions regarding this Privacy Policy, please contact us via the GitHub repository issues page or the Chrome Web Store support section.

# 隐私政策

**最后更新日期：2026年1月14日**

本隐私政策说明了“栈查查”（以下简称“本扩展”）如何收集、使用和处理您的信息。我们致力于保护您的隐私并确保透明的用户体验。

## 1. 数据收集与使用

**本扩展不收集、存储或传输任何个人信息。**

- **分析数据**：为了提供核心功能（分析 GitHub 仓库技术栈），本扩展会读取当前 GitHub 标签页的 URL，并从 GitHub 获取文件元数据和依赖文件（如 `package.json`, `go.mod`, `requirements.txt`）。
- **本地处理**：所有数据提取和分析均在您的浏览器**本地**完成。提取的任何数据都不会发送到任何外部服务器或第三方。
- **无个人数据**：我们不收集姓名、电子邮件地址、浏览历史或任何其他个人身份信息 (PII)。

## 2. 权限说明

为了正常运行，本扩展需要以下权限：

- **storage**：用于在浏览器中存储您的本地设置和偏好（如显示语言、UI 设置）。如果您开启了 Chrome 同步，这些数据将在您的设备间同步，但绝不会分享给我们。
- **主机权限 (`*://github.com/*`, `*://raw.githubusercontent.com/*`)**：用于从 GitHub 官方域名获取必要的仓库元数据和文件内容。

## 3. 数据安全

由于本扩展不收集数据，也不向外部服务器传输数据，您的数据在本地浏览器环境中是安全的。我们无法访问您分析的任何仓库或您的浏览活动。

## 4. 有限使用声明

本扩展对从 Google API 接收到的信息的使用和传输将遵守 [Chrome 网上应用店用户数据政策](https://developer.chrome.com/docs/webstore/program-policies/user-data-faq)，包括**有限使用**要求。

## 5. 政策更新

我们可能会不时更新本隐私政策，以反映我们做法的变化或出于法律原因。页面顶部的“最后更新日期”将说明最近一次更改的时间。

## 6. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过 GitHub 仓库的 Issues 页面或 Chrome 网上应用店的支持板块与我们联系。
